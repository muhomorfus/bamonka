Массив, в котором удалены элементы, которые являются полными квадратами

- 01-02 - обычные тесты;
- 03 - в массиве одно число, которое не является полным квадратом;
- 04 - обычный тест;

- 08-10 - обычные тесты.

#### Негативные тесты

- 01 - длина массива равна нулю;
